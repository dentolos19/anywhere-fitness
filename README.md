## 🧑‍💻 Contributing

### Prerequisites

- [Node.js](https://nodejs.org) 20+
- [pnpm](https://pnpm.io) 9+

### Installation

1. Clone this repository: `git clone https://github.com/dentolos19/anywhere-fitness.git`
2. Install dependencies: `pnpm install`
3. Run the server: `pnpm dev`
